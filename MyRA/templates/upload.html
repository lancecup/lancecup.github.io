How to Upload a Dataset in Python:

You can upload and load datasets programmatically in Python using the pandas library. Below are examples for common file formats:

1. CSV File:
# Import pandas
import pandas as pd

# Load a dataset from a CSV file
file_path = "path_to_your_file.csv"  # Replace with your file path
df = pd.read_csv(file_path)

# Display the first few rows of the dataset
print(df.head())
2. Excel File (.xlsx):
# Import pandas
import pandas as pd

# Load a dataset from an Excel file
file_path = "path_to_your_file.xlsx"  # Replace with your file path
df = pd.read_excel(file_path)

# Display the first few rows of the dataset
print(df.head())
3. Stata File (.dta):
# Import pandas
import pandas as pd

# Load a dataset from a Stata file
file_path = "path_to_your_file.dta"  # Replace with your file path
df = pd.read_stata(file_path)

# Display the first few rows of the dataset
print(df.head())

Replace path_to_your_file with the full file path to your dataset. These commands will load the data into a pandas DataFrame for further analysis.
